# <u>**Membrane keypad 1x4**</u>
<img src="img/keypad_4k.png" width="50%" > 

**Description**  
Membrane keypad with 4 keys (1 row x 4 columns).
When a key is pressed the key_n signal is connected to GND.

**Pinout**
* GND: common
* Key 1...4 : key pressed signals

**Software control**  
It requires 4 digital inputs with pull-up. When a button is pressed, it's signal is connected to GND, so the input logical status changes from 1 to 0.
