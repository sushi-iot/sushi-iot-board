# <u>**PCF8575**</u>
<img src="img/ioexp-mod.png" width="50%" > 

**Description**  
PCF8575 IO-Expander module. This module allows you to add an extra 16 GPIOs to the board. It communicates via the I2C interface.

**Pinout**
* GND
* VCC: 3.3-5V
* SDA: I2C data line
* SCL: I2C clock line
* INT: I2C interrupt

**Software control**  
The PCF8575 IO-Expander is controlled by the I2C data line.

**Sourcing**  
These modules are very common and can be found from different producers.
