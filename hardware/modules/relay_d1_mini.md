# <u>**Relay**</u>  
<img src="img/relay_d1_mini.png" width="50%" > 

**Description**  
"D1 mini" relay module from Wemos.
* Normally open contact 5A (250VAC/30VDC), 10A (125VAC), MAX: 1250VA/150W.
* Normally closed contact 3A (250VAC/30VDC), MAX: 750VA/90W.

**DOC**: [Wemos D1 Mini](https://www.wemos.cc/en/latest/d1_mini_shield/relay.html).

**Pinout**
Only 3 pin are used:
* G : GND
* 5V 
* D1 : ON/OFF 3.3v signal

**Software control**  
By a direct digital output (3.3v signal) it's possible control the D1 signal to close the relay contact.

**Sourcing**  
"D1 mini" module is avaible from the original producer or a lot of clone version. 
Any relay module that can be managed directly with a 3.3v from the micro can fit for the purpose.
