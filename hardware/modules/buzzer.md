# <u>**Buzzer**</u>  
![ACTIVE-BUZZER](img/buzzer.png)

**Description**  
3V Active Buzzer, it can produce different sounds for general purpose use.

**Pinout**
Only 3 pin are used:
* G : GND
* S[+] :  3.3v signal [ON, OFF, PWM]

**Software control**  
It can be managed by a digital output signal on S[+] pin with a PWM wave to modulate the frequency or simply with an ON/OFF signal.
